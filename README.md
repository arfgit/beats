# beats

Collaborative neon-synthwave beat sequencer. Vite + React 19 + TypeScript + Tailwind v4 + Zustand + Tone.js, Firebase-backed.

See [PLAN.md](./PLAN.md) for architecture and phase roadmap.

## Status

Phase 0 — scaffold.

## Dev

```bash
npm install
npm run dev
```

Firebase dev project: `beats-dev-ant` (display name "beats-dev").
